## 今回反映した主な修正

- 低圧動力は 3φ3W / 3φ4W のみ選択。
- 低圧電灯は 1φ2W / 1φ3W のみ選択。
- 開閉器裕度設定を追加。総負荷÷設定値を基準に推奨開閉器を選定。
- 「採用開閉器」を「推奨開閉器」へ文言変更。
- 推奨開閉器に合わせて幹線を自動選定。
- 必要最小開閉器と、必要最小開閉器に対する最小ケーブルサイズを表示。
- 自動推奨ケーブルと任意ケーブルサイズを分けて表示。
- 計算後に任意ケーブルサイズを変更し、再計算・判定可能。
- 計算するボタンをスクロール追従から通常配置へ変更。
- 設定画面に表示サイズ 70%〜130% を追加。文字サイズのみ変化し、レイアウト寸法は固定。
- 再起動時は履歴・保存済み・比較対象・表示サイズ以外の入力値を保持しない仕様へ変更。
- 保存済みは最大10件。同名保存は上書き。
- 技術資料に配管サイズ表を拡張。
- アプリ画面内ではメーカー名・商品名を表示しない方針で整理。
