## 今回の主な修正

- `app.js` の構文エラー原因だった `EFFICIENCY_OPTIONS` を修正。
- 低圧動力は 3φ3W / 3φ4W、低圧電灯は 1φ2W / 1φ3W に切り替え。
- 開閉器裕度設定 0.5〜1.0、標準 0.8 に対応。
- 必要最小開閉器、推奨開閉器、必要最小ケーブル、自動推奨ケーブル、任意ケーブルサイズを分離表示。
- ケーブル許容電流、合計概算質量、技術資料配管データ、保存・比較・CSV・Excel出力を実装。
- 再起動時は入力値を保持せず、履歴・保存済み・比較・表示サイズのみ保持。
- アプリ画面内と技術資料内では、特定メーカー名・商品名を表示しない構成。

## 注意

本アプリは参考計算用です。最終判断は内線規定、関係法令、現場条件、機器仕様、保護協調、短絡容量等を確認してください。
